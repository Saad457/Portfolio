## Projects

### EngineRay — AWS Serverless Cloud Application
*Python · TypeScript CDK · Lambda · Cognito · Route53 · ALB · CloudWatch · RDS · S3*

Individually designed and deployed a complete production-grade serverless architecture on AWS using CDK TypeScript. The entire infrastructure is provisioned through a single automated CI/CD pipeline triggered on each Git commit.

**Services used:**
- Amazon Cognito — user authentication, email verification, password policy
- Route53 + ACM — DNS routing and SSL/TLS certificate management
- Application Load Balancer — HTTPS traffic, health checks, security groups
- Auto Scaling Groups — EC2 instance scaling under varying load
- Python Lambda — data pipelines, RDS integration via pymysql, S3 event processing
- CloudWatch — log groups, metric filters, performance alarms
- AWS CDK (TypeScript) — full Infrastructure as Code deployment
